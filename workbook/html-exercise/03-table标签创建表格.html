首先嵌套在table标签里面的最基础的是tr (a row of cells in a table ) 代表每行数据,然后是 td (a cells of a table the contains data ) 可以理解为表格里每个单元格里的数据都需要用这个标签圈起来;这样两个基本元素就组 成的表格,但是为了更好的区分清楚,表头和内容,用thead包裹表头的tr,用tbody包裹 内容的tr;除此之外,表头tr里面的每个单元格数据,都用th包裹表示.
还有,可以在table里面添加标题(caption) 标签,注明标题.
